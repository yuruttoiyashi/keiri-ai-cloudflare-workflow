# Keiri AI Cloudflare Workflow

Cloudflare Workers AI を活用した、経理業務改善アプリのMVPです。
取引情報を入力すると、AIが勘定科目・消費税区分・リスクスコア・確認ポイントを自動判定し、保存前にミスや不自然な内容をチェックします。

経理担当者の入力ミス削減、監査前の確認作業の効率化、税理士・管理者によるチェック対象の絞り込みを目的とした業務支援アプリです。

---

## アプリ概要

日々の経理業務では、勘定科目の選択ミス、消費税区分の誤り、証憑内容との不一致などが発生しやすく、月次処理や監査前に多くの修正作業が発生します。

このアプリでは、入力段階でAIが内容を判定し、登録前に確認ポイントを提示することで、経理業務の「後から修正する」負担を減らすことを目指しています。

---

## 主な機能

### AI仕訳判定

取引先・摘要・金額・参加人数・メモなどの入力情報をもとに、AIが以下を推測します。

* 推奨勘定科目
* 推奨消費税区分
* AI信頼度
* リスクスコア
* 確認ポイント
* 要注意フラグ

---

### 保存前チェック

仕訳を保存する前に、AIがリスクや確認事項を表示します。

例：

* 消費税区分が未設定
* 会議費としては金額が高い
* 参加人数に対して金額が不自然
* 摘要が不足している
* 証憑確認が必要

---

### 監査キュー管理

要注意フラグが付いた取引は、監査キューとして確認しやすい形で管理できます。
すべての仕訳を確認するのではなく、AIが検知した取引を中心に確認できるため、監査前チェックの効率化につながります。

---

### D1データベース保存

Cloudflare D1 を使用して、仕訳データとAI判定結果を保存します。

保存対象の例：

* 日付
* 取引先
* 摘要
* 金額
* 勘定科目
* 消費税区分
* AI信頼度
* リスクスコア
* 確認ポイント
* 要注意フラグ

---

### CSV出力

登録した仕訳データはCSV形式で出力できます。
会計ソフトへの転記や、監査用資料の作成を想定しています。

## セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/ユーザー名/リポジトリ名.git
cd keiri-ai-cloudflare-workflow
```

---

### 2. パッケージをインストール

```bash
npm install
```

---

### 3. ビルド

```bash
npm run build
```

---

## Cloudflare設定

### 1. Cloudflareにログイン

```bash
npx wrangler login
```

---

### 2. D1データベースを作成

```bash
npm run d1:create
```

作成後に表示される `database_id` を `wrangler.toml` に設定します。

```toml
[[d1_databases]]
binding = "DB"
database_name = "keiri_ai_workflow"
database_id = "YOUR_D1_DATABASE_ID"
```

---

### 3. D1テーブルを作成

ローカル用：

```bash
npm run d1:local
```

本番用：

```bash
npm run d1:remote
```

## こだわったポイント

### 入力ミスを「登録前」に防ぐ設計

経理業務では、登録後にミスを修正するほど手戻りが大きくなります。
このアプリでは、保存前にAIが確認ポイントを提示することで、担当者自身がその場でミスに気づける流れを意識しました。

---

### 全件チェックではなく、要注意取引に絞る設計

管理者や税理士がすべての仕訳を確認するのではなく、AIがリスクの高い取引を抽出することで、確認作業を効率化できるようにしました。

---

### Cloudflareを活用した軽量なAI業務アプリ

Cloudflare Workers AI、Workers、D1を組み合わせることで、サーバーレスかつ軽量にAI機能付きの業務アプリを構築しています。
