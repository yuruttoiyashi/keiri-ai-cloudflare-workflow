# アプリ仕様メモ

## コンセプト

「入力後に直す」経理から、「登録前にAIが止める」経理へ。

## 資料から反映した要素

- AI自動入力
  - 日付・金額・取引先・摘要をもとに勘定科目と税区分を推定
- リアルタイム警告
  - 保存前に異常値や未入力を検出
- ピンポイント監査
  - 全件確認ではなく、要注意フラグ対象だけを重点確認
- Cloudflare構成
  - Pages: フロント
  - Workers AI: AI推論
  - D1: 仕訳データ保存
  - R2: 将来的な証憑画像・PDF保存先

## MVP機能

- 仕訳ドラフト作成
- AI推奨科目・税区分
- 保存前バリデーション
- 監査キュー
- CSV出力
- D1保存用API

## 将来拡張

- R2への証憑アップロード
- OCR / 画像解析
- 承認者ロール
- 操作ログ
- 月次監査レポート
- 会計ソフト連携CSVテンプレート
